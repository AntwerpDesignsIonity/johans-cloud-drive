# New Workspace Scaffold

This workspace contains a minimal Python project scaffold with a sample app and tests.

Quick start (PowerShell on Windows):

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r dev-requirements.txt
python -m src.app
pytest
```
